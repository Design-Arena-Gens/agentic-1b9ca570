Atelier Aura is a modern clothing ecommerce experience built with Next.js 14, the App Router, Tailwind CSS, and TypeScript. The storefront highlights capsule collections, storytelling content, and detailed product pages designed for a premium feel and efficient conversions.

## Tech Stack

- Next.js 14 (App Router)
- React 18 with Server & Client Components
- TypeScript
- Tailwind CSS
- Headless UI & Heroicons

## Features

- Editorial hero and curated product grids for key capsules
- Rich product detail pages with gallery, fabric notes, and related products
- Responsive, accessible layout optimized for desktop and mobile
- Newsletter sign-up interaction with optimistic success state
- Tailwind-powered design system with custom brand palette
